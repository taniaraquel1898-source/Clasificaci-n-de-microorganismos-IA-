# Clasificación de microorganismos con IA

Proyecto científico basado en inteligencia artificial para identificar microorganismos mediante secuencias de ADN y detectar mutaciones genéticas.

## Tecnologías utilizadas

- Python
- Streamlit
- Scikit-learn
- Biopython
- Machine Learning (Naive Bayes)

## Funciones

- Clasificación de microorganismos
- Análisis de secuencias de ADN
- Detección de mutaciones genéticas
- Visualización de resultados
